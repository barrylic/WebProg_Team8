Project Name:

Project 1 HTML & CSS, Yu-Gi-Oh Card Game Interface

Description:

For our project, we simulated the interface for the web card/video game Yu-Gi-Oh Duels! using only HTML and CSS. Our website allows the user to switch between players 1 and 2, see cards in play, flip cards, and link the original Yu-Gi-Oh duels site. This was all achieved using CSS tranforms, transitions, and animations.
